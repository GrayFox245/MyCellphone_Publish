1. Project Overview

A mobile-friendly web application to showcase Dov's digital art collection. The app allows visitors to browse, search, and filter artwork by name and themes, with automatic synchronization when new images are added.

Live URL

https://extraordinary-choux-2e83df.netlify.app

Key Features

  • Responsive image gallery (works on phone and desktop)
  • Search by image name
  • Filter by themes
  • Sort options: A-Z, Z-A, Newest First, Oldest First, Random
  • Click to view full image with description
  • Automatic sync from Google Drive
  • PWA support (can be installed on phone)

2. Architecture

[Your Computer]              [Google Cloud]                [Internet]
      |                            |                           |
Images Folder  --auto-sync-->  Google Drive Folder  <--reads-- Web App (Netlify)
Excel File     --auto-sync-->  Google Drive                    |
                                   |                           |
                            Google Apps Script          User's Phone/Browser
                            (converts to JSON)

How It Works

  1. Images and Excel file sync to Google Drive (via Google Drive desktop app)
  2. Google Apps Script reads the spreadsheet every 5 minutes
  3. Script generates image-data.json with image URLs and metadata
  4. Web app fetches this JSON and displays the gallery

5. How to Add New Images

  1. Add the image file to your local "All my images small" folder (it will sync to Google Drive automatically)
  2. Update the spreadsheet "Description and themes of my images" with:
    • Image Name (must match filename without extension)
    • Description
    • Themes (comma-separated)
    • Date Created
  3. Wait 5 minutes for the Google Apps Script to update the JSON
  4. Download the new image-data.json from Google Drive
  5. Re-deploy to Netlify by dragging the folder to Netlify
Note: The image name in the spreadsheet must exactly match the filename (without .jpg extension). For example, if the file is "Sunset Beach.jpg", the spreadsheet should have "Sunset Beach".

6. Troubleshooting

Image not showing?

  • Check that the name in spreadsheet matches the filename exactly
  • Make sure the Google Drive folder is shared publicly ("Anyone with the link")
  • Run the Google Apps Script manually and check for errors

Script errors?

  • Verify the FOLDER_ID in the script matches your folder
  • Make sure the spreadsheet is in the same folder as the images
  • Check that the spreadsheet is a Google Sheets file (not .xlsx)

  • Themes: "Themes", "Theme", or "themes"
  • Date: "Date Created", "Date", "Created", or "date created"

Supported Image Formats

jpg, jpeg, png, gif, webp, bmp, tiff

Google Apps Script Trigger Setup

  1. Go to script.google.com
  2. Open your "DovArt Sync" project
  3. Click on "Triggers" (clock icon) in the left sidebar
  4. Click "+ Add Trigger"
  5. Settings:
    • Function: generateJSON
    • Event source: Time-driven
    • Type: Minutes timer
    • Interval: Every 5 minutes
  6. Click Save
